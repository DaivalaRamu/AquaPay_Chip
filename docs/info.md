# AquaPay Chip

## Project Description

AquaPay Chip is a digital logic design implemented for TinyTapeout.  
This project processes input signals and generates corresponding outputs based on predefined logic conditions.

The goal of this design is to demonstrate a simple hardware-based decision or control system.

---

## How it works

The design takes input signals through input pins and processes them using combinational/sequential logic.

- Inputs are fed into logic blocks
- Internal logic evaluates conditions
- Outputs are generated based on logic rules

The circuit may include:
- Logic gates
- Flip-flops (if sequential)
- Control signals

The output changes according to the applied input conditions.

## How to test

You can test the design using simulation tools.

### Steps:

1. Provide input signals to the design
2. Run simulation using a Verilog simulator
3. Observe output signals
4. Compare output with expected results

### Tools you can use:
- Icarus Verilog (iverilog)
- GTKWave

### Example:

- Apply different input combinations
- Check waveform output
- Verify correct logic behavior
